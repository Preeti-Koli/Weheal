# Weheal
WeHeal Chatbot is an AI-powered conversational assistant that provides emotional support and well-being guidance. It uses NLP and sentiment analysis to understand user input and deliver supportive responses, including stress relief tips and positive affirmations in a safe environment.

AI-powered conversational assistant for emotional support and well-being guidance
Provides a safe, non-judgmental space for users to express thoughts and feelings
Uses Natural Language Processing (NLP) to understand user input
Performs sentiment analysis to detect user emotions (happy, sad, stressed, etc.)
Generates supportive responses, including calming tips and positive affirmations
Helps users with stress management and daily self-care suggestions
Built using Python with libraries like NLTK / TextBlob
Interface developed using Flask / Tkinter
Can be extended with voice interaction (speech-to-text & text-to-speech)
Designed as a supportive tool, not a replacement for professional help
